## Insta Schooler Project ##

To run the project :
- Install Docker Desktop
- clone Repo
- Open a new Terminal :
    - run npm install -g @nestjs/cli
    - nvm install 18.14
    - nvm use 18.14
    - node -v // to verify version 18
    - run npm install -g @angular/cli@^16.0.0 //version 16
    - docker-compose up --build
